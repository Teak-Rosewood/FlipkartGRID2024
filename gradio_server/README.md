# Gradio Server

This repository contains the Gradio Server project.

## Installation

Requires python 3.10+ (gradio) 
To install the necessary dependencies, run the following command: (NOTE: edit paddle link for your GPU configuration)

```bash
pip install -r requirements.txt
```

## Usage

To start the server, use the following command:

```bash
python run.py
```
To import API

```
from api import *
```
